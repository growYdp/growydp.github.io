    # 原始值转布尔

    在JavaScript中，只有6种值可以被转换成false，其他都会被转换成true。+0、-0、NaN、''、null、undefined、false

    使用Boolean函数将类型转换成布尔类型

    console.log(Boolean()) // false
console.log(Boolean(false)) // false
console.log(Boolean('')) // false
console.log(Boolean(NaN)) // false
console.log(Boolean(+0)) // false
console.log(Boolean(-0)) // false
console.log(Boolean(undefined)) // false
console.log(Boolean(null)) // false

    注意点：

    • Boolean函数不传任何参数时，会返回false

    # 原始值转数字

    # Number

    使用Number函数将类型转换成数字类型，如果参数无法被转换为数字，返回NaN。

    es5 Number (opens new window)

    Number函数底层其实就是调用ToNumber(value)计算的数字值（不是数字对象），如果没提供值，返回+0。

    • Symbol类型的值不能转换为数字，会报错。

    # ToNumber

    参数类型 结果
    Undefined NaN
    Null +0
    Boolean 如果参数是true，结果是1。如果参数是false，结果是+0
    Number 结果是传入的参数（没有转换）
    String 比较复杂，转换过程如下
    // 如果没给Number函数传参 返回+0
console.log(Number()) // +0

// undefined 和 null
console.log(Number(undefined)) // NaN
console.log(Number(null)) // +0

// boolean
console.log(Number(false)) // +0
console.log(Number(true)) // 1

// string
console.log(Number("123")) // 123
console.log(Number("-123")) // -123
console.log(Number("1.2")) // 1.2
console.log(Number("000123")) // 123
console.log(Number("-000123")) // -123

console.log(Number("0x11")) // 17

console.log(Number("")) // 0
console.log(Number(" ")) // 0

console.log(Number("123 123")) // NaN
console.log(Number("foo")) // NaN
console.log(Number("100a")) // NaN

    如果通过Number转换函数传入一个字符串

    • 会试图将其转换成一个整数或浮点数
    • 会忽略所有前导的0
    • 如果有一个字符不是数字，结果都会返回NaN

    # parseInt/parseFloat

    这两个方法专注于字符串是否包含数值模式。

    • 字符串最前面的空格会被忽略，从第一个非空格字符开始转换。尽可能解析更多的数值字符
    • 如果第一个非空格字符串不是数值字符、正负号，立即返回NaN
    • Number函数遇到空字符串返回0，它们遇到空字符串返回NaN
    • 依次检测每个字符，直到字符串末尾或者遇到第一个不是数值的字符。
    • 如果0x开头，会被解释为十六进制整数，如果0开头后面跟着数值字符，非严格模式下被解释为八进制整数。
    // parseInt
console.log(parseInt('1 a')) // 1
console.log(parseInt('     a1')) // NaN
console.log(parseInt('  222')) // 222
console.log(parseInt('0.11')) // 0
console.log(parseInt('0xFF')) // 255

// parseFloat
console.log(parseFloat('3.1 a')) // 3.1
console.log(parseFloat('.2')) // 0.2

    不同

    parseInt可以接收第二个参数，用于指定底数（进制数）

    • parseFloat只解析十进制值，不能指定底数，所以十六进制数值始终返回0
    • 如果字符串表示整数，没有小数点或者小数点后面只有一个0，parseFloat返回整数

    # 原始值转字符

    使用String函数将类型转换成字符串类型

    # String

    返回由ToString(value)计算的字符串值（不是字符串对象），如果没有提供值，则返回空字符串。

    String可以转化undefined和null，但是toString不可以

    # ToString

    参数类型 结果
    Undefined "undefined"
    Null "null"
    Boolean 如果参数是true，返回“true”。参数为false，返回“false”
    Number 复杂，看例子
    String 返回与之相等的值

    es5 ToString (opens new window)

    • 如果传入NaN，返回NaN字符串
    • 如果传入+0或者-0，返回字符串0
    • 如果传入值小于0，返回负号并且将参数调用ToString(-参数)的连接
    • 如果传入参数是正负无穷，返回Infinity字符串
    console.log(String()) // ''
console.log(String(undefined)) // 'undefined'
console.log(String(null)) // 'null'
console.log(String(true)) // 'true'
console.log(String(false)) // 'false'

console.log(String(0)) // '0'
console.log(String(-0)) // '0'
console.log(String(NaN)) // 'NaN'
console.log(String(Infinity)) // 'Infinity'
console.log(String(-Infinity)) // '-Infinity'
console.log(String(1)) // '1'
console.log(String(-7)) // '-7'

    切记，和上面的ToString不一样（其是底层内置方法）。

    返回当前值的字符串等价物，几乎所有值都有toString()方法

    记忆方法：只要有类型构造函数，就可以使用这个方法，null和undefined没有这个方法！！！调用会报错（Uncaught TypeError: Cannnot read properties of undefined (reading 'toString')）

    # 原始值转对象

    原始值转换为对象，原始值只需要通过调用String()、Number()或者Boolean()构造函数，转换为它们各自的包装对象。

    红宝书 原始值包装类型 (opens new window)

    基本类型是怎么能使用方法的呢？？？

    let s1 = "some text";
let s2 = s1.substring(2);

    s1是一个包含字符串的变量，是一个原始值。然后在这个原始值上调用了substring方法，把结果存储到了s2中。

    原始值不是对象，不应该有方法，但是竟然没报错。因为后台进行了处理，当访问s1时，是以读模式访问的，就是从内存中读取变量保存的值

    在以读模式访问字符串的时候，会执行以下3步：

    1. 创建一个String类型的实例
    2. 调用实例上的特定方法
    3. 销毁实例
    let s1 = new String("some text");
let s2 = s1.substring(2);
s1 = null;

    const a = 1
console.log(typeof a) // number
const b = new Number(a)
console.log(typeof b) // object

    # 对象转布尔值

    对象到布尔值的转换：对于对象（包括数组和函数）都转换为true，对于包装对象也一样。

    console.log(Boolean(new Boolean(false))) // true

    # 对象转字符串和数字

    对象到字符串和对象到数字的转换都是通过调用待转换对象的一个方法来完成的。

    JavaScript对象有两个不同的方法来执行转换：toString和valueOf。这两个方法是真实暴露出来的。

    # 隐式类型转换

    console.log(1 + '1') // 11

    在JavaScript中，两个类型不同的值可以运行。JavaScript自动的将数据类型进行了转换，这种转换叫做隐式类型转换。

    # 一元运算符+

    console.log(+'1')

    es5 一元+运算符 (opens new window)

    • 一元+运算符将其操作数转换为数字类型
    • 转换步骤如下：
      1. 让 expr 作为运算符求值的结果。
      2. ToNumber(GetValue(expr))
