    # Ajax Fetch Axios 三者有什么区别

    三种都是网络请求，但是维度不一样

    • Ajax（Asynchronous JavaScript and XML），一种技术统称
    • Fetch (opens new window) 一个具体的API 浏览器原生API，用于网络请求
    • Axios 第三方库

    库和API的区别：库有成千上万，但是API很少（XMLHttpRequest和fetch），实际项目中还是使用现成库

    # 使用XMLHttpRequest实现Ajax

    XMLHttpRequest (opens new window)

    function ajax1(url, callback) {
  const xhr = new XMLHttpRequest()
  xhr.onreadystatechange = function() {
    if (xhr.readyState === 4) {
      if (xhr.status === 200) {
        callback(xhr.responseText)
      }
    }
  }
  xhr.open('GET', url, false)
  xhr.send(null)
}

    # 使用fetch发送请求

    function ajax2(url) {
  return fetch(url).then(res => res.json())
}

    # 防抖

    • 防止抖动 你先抖动着，啥时候停了 再执行下一步
    • 一个搜索输入框，等输入停止之后，再触发搜索
    function debounce(fn, delay = 200) {
  let timer
  return function(...args) {
    if (timer) {
      clearTimeout(timer)
    }
    timer = setTimeout(() => {
      fn.apply(this, args)
    }, delay)
  }
}

    # 节流

    • 节省交互沟通。流：不一定指流量
    • 别急 一个一个来 按事件节奏来 插队者无效
    // flag
function throttle(fn, delay) {
  let flag = true
  return function(...args) {
    if (!flag) {
      return
    }
    flag = false
    setTimeout(() => {
      flag = true
      fn.apply(this, args)
    })
  }
}

// time
function throttle1(fn, delay) {
  let startTime = new Date()
  return function(...args) {
    let endTime = new Date()
    if ((endTime - startTime) > delay) {
      fn.apply(this, args)
      startTime = endTime
    } else {
      return
    }
  }
}

    # for...in 和 for...of 有什么区别？

    适用于不同的数据类型

    • 遍历对象：for...in可以，for...of不可以

    • 遍历Map Set：for...of可以，for...in不可以

    • 遍历generator：for...of可以，for...in不可以

    • for...in用于可枚举数据 如对象 数组 字符串，遍历得到key

    • for...of同于可迭代数据 数组 字符串 Map Set，遍历得到value

    # px % em rem vw vh 有什么区别

    # px %

    • px是基本单位 绝对单位（其他的都是相对单位）
    • % 相对于父元素的比例

    # em rem

    • em 相对于当前元素的 font-size（好多文档说相对于父元素 待考察）
    • rem 相对于根节点的 font-size
    @media only screen and (max-width: 374px) {
  /* iphone5 或者更小尺寸 */
  html {
    font-size: 86px
  }
}

@media only screen and (min-width: 375px) and (max-width: 413px) {
  /* iphone6/7/8/x */
  html {
    font-size: 100px
  }
}

@media only screen and (min-width: 414px) {
  /* iphone6p 或者更大尺寸 */
  html {
    font-size: 110px
  }
}

    # vw vh

    • vw 屏幕宽度的 1%
    • vh 屏幕宽度的 1%
    • vmin 上面两个值的最小值 vmax 上面两个值的最大值
